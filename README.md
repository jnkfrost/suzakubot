Bot WhatsApp Gruppo Anime
Setup
Installa Node.js (versione 16 o superiore)
Apri terminale nella cartella del progetto
Esegui:
npm install
Avvia il bot:
npm start
Scansiona il QR code mostrato nel terminale con l'app WhatsApp (Impostazioni > Dispositivi collegati > Collega un dispositivo)
Comandi disponibili nel gruppo:
!aiuto
!regole
!citazione
!sondaggio
